NewRelic Instrumentation for SlackRubyBot

NewRelic instrumentation for [slack-ruby-bot](https://github.com/slack-ruby/slack-ruby-bot).

![](screenshots/breakdown.png)

## Install

Add to Gemfile.

```ruby
gem 'newrelic-slack-ruby-bot'
```
